Help

This is a procedurally generated maze. An AI agent has been trained with Q-learning to solve these kinds of mazes.

There is also an alternative to play the game with manual controls. Press the following keys to switch between the different game modes:


A : AI mode. Watch the pre-trained agent solve the current maze. (Default)

P : Manual player mode. Press 'I' to show controls.

N : Start a training session for a new AI agent.

Instructions

Use the arrow keys to move the ball and find your way out of the maze before your energy is gone.

You lose some energy for every step you take. Note that brick walls are solid but bushes are not.

However, if you go through a bush, more energy is spent. The darker the bush the more energy it requires.

Good luck!

AI agent mode

This mode will automatically play a game with the newest agent.

Else chose a number between [0, 0] to play another agent.

To start a new training session for a new agent click 'N'.

To continue the training session for this agent click 'C'.

To switch to manual player mode click 'P'.

To speed up training session press '+'

To slow down the training session press '-'

Manual player mode

In this mode the balls move from user input.

To show instructions press 'I'

To switch to AI agent mode click 'A'.
